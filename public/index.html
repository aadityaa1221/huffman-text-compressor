⚠️
macOS TextEdit users: save as plain text first TextEdit saves as RTF by default. Your test.txt is actually an RTF file — the compressor works perfectly, but the restored file looks wrong because macOS opens it differently. Fix: in TextEdit go to Format → Make Plain Text, then save. Or use a plain-text editor like VS Code, Sublime, or nano.
